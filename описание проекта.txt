TCP Клиент:
открывает текстовый файл d:\test.txt вида:

1111 sdgdfg
22222 dfgdf
3333 dfg
444 sdfsdfsdf
5555 sdfsdfsdf

читает из него строки. Парсит строки на два параметра - целое число и строка.
Формирует из этих параметров массив байтов и отправляет сообщение серверу содержащее:
1)длину сообщения
2)1-й параметр
3)2-й параметр



TCP Сервер:
Ожидает подключений.
Принимает поток байтов от клиента. Разбивает поток на отдельные сообщения используя длину.
Вставляет в таблицу test_tb базы данных test_bd строку с 1-м числовым и 2-м символьным параметром.



Таблица test_tb:
Id(INT ключевое);   num(int);   name(varchar);




Клиент и сервер реализованы на ".net" visual studio express 2012
Сервер многопоточный. Средствами ThreadPool
База данных MySql
Работа с базой данных через библиотеку ADO.NET Driver for MySQL (Connector/NET)
